# 🛠️ Albertool Tools

## 📱 Sobre o App

**Albertool Tools** é um aplicativo de utilidades e ferramentas financeiras desenvolvido com React Native e TypeScript. O app oferece consultas a APIs em tempo real, simuladores financeiros e ferramentas do dia a dia.

## ✨ Funcionalidades

### 📈 APIs Financeiras
- **Bitcoin** - Cotação atual via CoinGecko
- **Dólar** - Cotação via Banco Central do Brasil
- **Euro** - Cotação via AwesomeAPI
- **Taxa Selic** - Dados oficiais do BCB

### 🌤️ Utilidades
- **Previsão do Tempo** - Clima atual e previsão (Open-Meteo)
- **Monitor do Rio** - Simulação de subida/descida de nível
- **Conversor de Tempo** - Anos para dias, meses e horas

### 💰 Calculadoras Financeiras
- **Juros Compostos** - Simulação com CDB/LCI/Tesouro
- **Valor por Hora** - Calcule quanto vale sua hora trabalhada
- **Horas Trabalhadas** - Cálculo de horas extras
- **Simular FIIs** - Proventos de Fundos Imobiliários

### 🛠️ Outras Ferramentas
- **Reajuste Salarial** - Cálculo percentual
- **Contador de Moedas** - Soma valores em reais

## 📱 Tecnologias

- React Native 0.81.5
- TypeScript 5.9.2
- Expo 52.0.0
- React Navigation 7.x
